---
name: create-branch
description: How to create and name git branches
---

Please create branches with the following template: gonza/<branch-name>

If I have not given to you the branch name yet, please ask for it. If I accidentally give you the full branch name (i.e gonza/foo-bar), please discard the gonza/ part of it. Let's not end up with a branch called gonza/gonza/foo-bar
